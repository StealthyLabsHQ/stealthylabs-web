Caveman
Lang

Retour a la documentation

1 Presentation

Caveman Lang est un protocole de survie extreme des tokens qui reduit de 60 a 90 % les sorties superflues des LLM. Il definit des niveaux de compression et des regles de redaction pour Claude, ChatGPT, Gemini, et tout LLM prenant en charge la configuration du prompt systeme.

Le protocole supprime les formules creuses, les articles redondants, les confirmations verbeuses et tout contenu a faible valeur informative.

2 Niveaux de compression

Niveau Intensite Description
L0 Aucune Pas de compression - sortie prose complete
L1 Legere Suppression des formules creuses uniquement
L2 Moderee Suppression des articles + abreviations
L3 Defaut (~75%) Jeu de regles complet - pas de remplissage, pas d'articles, symboles, defaultdrop, saliencek K=3
L4 Extreme Compression maximale - sortie concise, ponctuation minimale
Defaut : L3 est le niveau recommande pour la plupart des flux de travail de developpement.

3 Regles L3

Le niveau L3 applique les regles suivantes a toutes les sorties :

Regles de prose

  • Pas de remplissage - jamais "Certainement !", "Je serais ravi de", "J'espere que cela vous aide"
  • Pas de repetition - ne jamais reformuler la question avant de repondre
  • Pas d'articles - supprimer le/la/les/un/une sauf si l'omission cree une ambiguite
  • Abreviations - utiliser les formes courtes standard : err cfg req res usr fn ctx srv dep env msg auth
  • Symboles - utiliser + ~ = @ # ! a la place des mots quand c'est clair

Defaultdrop

Omettre les valeurs OK/par defaut/attendues. N'emettre que les ecarts par rapport a la norme. Si une valeur est standard ou inchangee, la passer entierement sous silence.

Saliencek K=3

Afficher 3 elements maximum. Si d'autres existent, ajouter "+N autres" plutot que tout lister. Prioriser par pertinence et impact.

4 Utilisation

Ajouter au CLAUDE.md

Coller les regles Caveman Lang dans votre ~/.claude/CLAUDE.md ou dans le CLAUDE.md du projet :

# Caveman Mode - Token Survival Protocol (always active)
Default tier: L3 (~75% compression).
Override per message: L0/L1/L2/L3/L4.

Installer comme skill Claude Code

  1. Ouvrir Claude Code
  2. Executer /skills install caveman-lang
  3. Le skill s'active automatiquement dans toutes les sessions

ChatGPT / prompt systeme

Coller le jeu de regles L3 dans vos instructions personnalisees ChatGPT ou dans le prompt systeme de tout appel API. Compatible avec tout LLM prenant en charge la configuration du prompt systeme.

5 Compatibilite

  • Claude - skill Claude Code ou CLAUDE.md
  • ChatGPT - instructions personnalisees ou prompt systeme
  • Gemini - champ d'instruction systeme
  • Tout LLM - tout modele avec support du prompt systeme
