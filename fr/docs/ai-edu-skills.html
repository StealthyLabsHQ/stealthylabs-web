1 Presentation

AI Edu Skills est une collection de skills pour ChatGPT et Claude axes sur le soutien academique et le tutorat. Il couvre l'ensemble du flux de travail etudiant, de la recherche et de la redaction jusqu'a la preparation aux examens et la gestion des citations.

Concu pour les etudiants du lycee a l'universite, chaque skill fournit des conseils structures adaptes aux standards academiques.

2 Fonctionnalites

  • Aide a la recherche - trouver et evaluer les sources, structurer les revues de litterature
  • Redaction de dissertations - plans, argumentation, transitions, conseils pour la conclusion
  • Preparation aux examens - questions de pratique, revision des concepts, techniques de memorisation
  • Planification des revisions - calendriers de revision, gestion des priorites, suivi des echeances
  • Aide aux citations - mise en forme APA, MLA, Chicago et gestion de bibliographie

3 Installation

Skill Claude Code

  1. Ouvrir Claude Code
  2. Executer /skills install ai-edu-skills
  3. Les skills s'activent dans la session en cours

Instructions personnalisees ChatGPT

  1. Ouvrir le depot GitHub
  2. Copier le prompt du skill souhaite
  3. Coller dans les instructions personnalisees ChatGPT ou dans le prompt systeme d'un nouveau GPT

4 Public cible

  • Lyceens - redaction, preparation aux examens, soutien scolaire
  • Etudiants en licence - dissertations, citations, redaction academique
  • Etudiants en master/doctorat - revues de litterature, soutien a la these, planification

5 Prerequis

  • Claude - Claude Code avec support des skills, ou Claude.ai
  • ChatGPT - ChatGPT Plus, Team, ou acces API
  • Aucune dependance supplementaire requise
