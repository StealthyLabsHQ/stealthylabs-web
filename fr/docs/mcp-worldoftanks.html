1 Presentation

mcp-worldoftanks est un serveur MCP TypeScript qui connecte les CLI d'IA a l'API Wargaming.net, permettant des requetes en langage naturel sur les statistiques de compte World of Tanks. Interrogez votre assistant IA pour recuperer des profils joueurs, des performances par char ou l'historique de batailles directement depuis votre terminal.

Compatible avec Claude Code, Claude Desktop, Gemini CLI et tout client IA compatible MCP.

2 Fonctionnalites

  • Statistiques de compte - taux de victoire global, degats, WN8 et plus
  • Statistiques par char - analyse des performances par vehicule
  • Profils joueurs - details du compte et informations de clan
  • Historique de batailles - donnees de session recentes
  • Multi-clients - fonctionne avec tout CLI IA compatible MCP
  • API Wargaming.net - source officielle, toutes regions supportees

3 Installation

Option 1 - npx (sans build)

Ajoutez directement dans votre configuration MCP via npx. Voir la section Configuration ci-dessous.

Option 2 - Depuis les sources

Necessite Node.js 18+.

git clone https://github.com/StealthyLabsHQ/mcp-worldoftanks.git
cd mcp-worldoftanks
npm install
npm run build

4 Configuration

Definissez votre cle API Wargaming.net comme variable d'environnement : 

export WG_API_KEY=votre_cle_api

Ajoutez le serveur a votre fichier de configuration MCP. Via npx : 

{
  "mcpServers": {
    "mcp-worldoftanks": {
      "command": "npx",
      "args": ["-y", "@stealthylabs/mcp-worldoftanks"],
      "env": {
        "WG_API_KEY": "votre_cle_api"
      }
    }
  }
}

Ou si installe depuis les sources, pointez vers le build :

{
  "mcpServers": {
    "mcp-worldoftanks": {
      "command": "node",
      "args": ["/chemin/vers/mcp-worldoftanks/dist/index.js"],
      "env": {
        "WG_API_KEY": "votre_cle_api"
      }
    }
  }
}
Cle API : Obtenez une cle API Wargaming.net gratuitement sur developers.wargaming.net.

5 Prerequis
