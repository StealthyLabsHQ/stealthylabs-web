Specter
Discord Bot

Retour aux docs

1 Aperçu

Specter est un bot Discord modulaire développé en TypeScript avec discord.js v14, combinant lecture musicale, commandes IA (Claude + Gemini), génération d'images, modération de serveur et outils utilitaires. Il propose 61 commandes slash réparties sur 6 modules indépendants.

SQLite gère toute la persistance : configuration des guildes, préférences utilisateurs, historique des conversations et reprise de la file musicale après redémarrage. Version 2.0.0.

2 Modules

2.1 Musique (18 commandes)

Lecteur musical complet prenant en charge YouTube, Spotify et SoundCloud via le backend yt-dlp. La file d'attente persiste entre les redémarrages du bot. Filtres audio : bassboost, nightcore, vaporwave, 8d.

2.2 IA (8 commandes)

Système dual fournisseur : Claude (Anthropic) et Gemini (Google). Le choix de modèle par utilisateur est stocké dans SQLite. L'historique de conversation conserve les 25 derniers messages par utilisateur par guilde. La limitation de débit applique un délai de base de 8s, extensible à 60s si l'API est surchargée. Des limites d'utilisation journalières s'appliquent par modèle.

Commande Description
/ask [question] [provider] [lang]Interroger Claude ou Gemini avec sélection de langue
/summarize [messages] [provider]Résumer les messages récents d'un salon
/nanobanana [prompt] [style] [ratio] [resolution]Générer des images avec Gemini
/setmodel [model]Choisir son modèle IA personnel (variantes Claude ou Gemini)
/translate [language] [text]Traduction par IA
/roast [user]Roast humoristique d'un utilisateur par l'IA
/vision [image] [prompt]Analyser une image avec l'IA
/catchupRésumer le contexte de conversation récent

Modèles disponibles :

  • Claude : Sonnet 4.6 (recommandé), Opus 4.6 (plus puissant), Haiku 4.5 (plus rapide)
  • Gemini : Flash 3 Preview (recommandé), Pro 3.1 Preview (plus puissant), Flash Lite 3.1 (moins cher)

Génération d'images (/nanobanana) :

  • Styles : photorealistic, anime, cartoon, oil painting, sketch, pixel art, cinematic
  • Ratios : 1:1, 16:9, 9:16, 4:3, 3:4
  • Résolutions : 512px, 1K, 2K, 4K (limites journalières par tier)
  • Le délai adaptatif augmente lorsque l'API est surchargée

3 Fonctionnalités automatiques

Comportements déclenchés par des événements, sans commande slash :

  • Auto-rôle - attribuer un rôle à chaque nouveau membre (requiert /config autorole)
  • Messages de bienvenue - accueillir les nouveaux membres dans un salon désigné
  • Salons vocaux temporaires - rejoindre un salon hub crée un salon vocal personnel, supprimé lorsqu'il est vide
  • Journal des messages - les messages supprimés et modifiés sont consignés dans le salon de modération
  • Auto-modération IA - analyse optionnelle du contenu par Gemini avec délai anti-spam de 5s par utilisateur, consigne le contenu signalé

4 Installation

  1. Cloner le dépôt et entrer dans le répertoire : 
    git clone <repository-url>
cd discord-bot
  2. Installer les dépendances : 
    npm install
  3. Copier le modèle d'environnement : 
    cp .env.example .env
  4. Modifier .env avec vos tokens et clés API.
  5. Déployer les commandes slash sur Discord : 
    npm run deploy
  6. Démarrer en mode développement : 
    npm run dev
    Ou compiler et lancer en production : 
    npm run build
npm run start

Production avec PM2

npm run build
npm run deploy
pm2 start dist/index.js --name specter
pm2 save

5 Configuration

Variables d'environnement

Requises :

Variable Description
DISCORD_TOKENToken du bot depuis le Portail Développeur Discord
CLIENT_IDID de l'application depuis le Portail Développeur Discord

Optionnelles :

Variable Description
GUILD_IDID du serveur de développement pour un déploiement instantané des commandes
ANTHROPIC_API_KEYAccès direct à l'API Claude (module IA désactivé si absent)
GOOGLE_AI_API_KEYAPI Gemini + génération d'images /nanobanana
GOOGLE_CLOUD_PROJECTID du projet GCP pour Vertex AI Claude
GOOGLE_CLOUD_REGIONRégion GCP pour Vertex AI (défaut : us-east5)
GOOGLE_APPLICATION_CREDENTIALSChemin vers le fichier JSON du compte de service GCP
SPOTIFY_CLIENT_IDRésolution des liens Spotify
SPOTIFY_CLIENT_SECRETRésolution des liens Spotify
NODE_ENVdevelopment ou production

Portail Développeur Discord

Paramètres requis dans le Portail Développeur Discord :

  • Intents privilégiés : Server Members, Message Content, Guild Members, Guild Presences
  • Scopes du bot : bot, applications.commands
  • Permissions recommandées : Administrateur, ou granulaires : Gérer les rôles, Gérer les salons, Expulser/Bannir des membres, Modérer des membres, Gérer les messages
Note : Placez un fichier cookies.txt (format Netscape) à la racine du projet pour contourner la détection de bot de YouTube pour la musique.

6 Prérequis

  • Node.js 20+
  • FFmpeg - installation système recommandée ; ffmpeg-static utilisé en secours
  • Au minimum DISCORD_TOKEN + CLIENT_ID pour démarrer - les modules IA et musique se dégradent gracieusement sans les clés API optionnelles
