Specter
Discord Bot

Retour aux docs

1 Apercu

Specter est un bot Discord modulaire (v2.0.0) développé en TypeScript avec discord.js v14. Il propose 74 commandes slash réparties sur 6 modules indépendants : lecteur musical, assistant IA, modération, utilitaires, fun et configuration du serveur.

Toute la persistance est assurée par SQLite (8 tables) : configuration des guildes, préférences utilisateurs, historique des conversations, avertissements et reprise de la file musicale après redémarrage. Prend en charge les commandes slash et le préfixe texte (.commande ou specter commande).

2 Modules

2.2 IA (9 commandes)

Système dual fournisseur : Claude (Anthropic) et Gemini (Google). Le choix de modèle par utilisateur est stocké dans SQLite. L'historique de conversation conserve les 25 derniers messages par utilisateur par guilde. La limitation de débit applique un délai de base de 8s, extensible à 60s si l'API est surchargée. Des limites d'utilisation journalières s'appliquent par modèle.

Commande Description
/ask [question] [provider] [lang]Interroger Claude ou Gemini
/summarize [messages] [provider]Résumer les messages récents d'un salon
/tldr [url]Résumer une page web
/nanobanana [prompt] [style] [ratio] [resolution]Générer des images avec Gemini
/setmodel [model]Choisir son modèle IA personnel
/translate [language] [text]Traduction par IA
/roast [user]Roast humoristique basé sur les messages récents de l'utilisateur
/vision [image] [prompt]Analyser une image avec l'IA
/catchupRésumer le contexte de conversation récent

Modèles disponibles :

  • Claude : Sonnet 4.6 (recommandé), Opus 4.6 (plus puissant), Haiku 4.5 (plus rapide)
  • Gemini : Flash 3 Preview (recommandé), Pro 3.1 Preview (plus puissant), Flash Lite 3.1 (moins cher)

Génération d'images (/nanobanana) :

  • Styles : photorealistic, anime, cartoon, oil painting, sketch, pixel art, cinematic
  • Ratios : 1:1, 16:9, 9:16, 4:3, 3:4
  • Résolutions : 512px, 1K, 2K, 4K (limites journalières par tier)
  • Le délai adaptatif augmente lorsque l'API est surchargée

3 Fonctionnalites automatiques

Comportements déclenchés par des événements, sans commande slash :

  • Auto-rôle - attribue un rôle à chaque nouveau membre via /config autorole
  • Messages de bienvenue - accueille les nouveaux membres dans un salon désigné
  • Salons vocaux temporaires - rejoindre un salon hub crée un salon vocal personnel, supprimé lorsqu'il est vide
  • Journal des messages - les messages supprimés et modifiés sont consignés dans le salon de modération
  • Auto-modération IA - analyse optionnelle du contenu par Gemini avec délai de 5s par utilisateur, signale et consigne le contenu suspect
  • Snipe de messages - conserve le dernier message supprimé ou modifié par salon

4 Commandes par prefixe texte

Toutes les commandes fonctionnent aussi avec un préfixe texte, en plus des commandes slash :

  • Format : .commande [args] ou specter commande [args]
  • Exemple : .play never gonna give you up ou specter ask what is rust
  • Le préfixe et le nom du bot sont configurables via .env
