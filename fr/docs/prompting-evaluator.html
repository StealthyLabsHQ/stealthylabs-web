1 Vue d'ensemble

Prompting Evaluator est un outil professionnel de prompt engineering concu pour evaluer, noter et optimiser des instructions IA. Il supporte plusieurs backends - modeles locaux via Ollama et APIs cloud incluant Google Gemini et Anthropic Claude.

L'outil propose une interface en ligne de commande et une interface web, adapte aussi bien a l'iteration rapide en terminal qu'aux workflows d'evaluation structures.

2 Fonctionnalites

  • Evaluation multi-modeles - executer des prompts sur Ollama, Gemini et Claude simultanement
  • Rubriques de notation - criteres structures pour une evaluation coherente des prompts
  • Comparaison de prompts - analyse cote a cote de variantes de prompts
  • Suggestions d'optimisation - recommandations concretes pour ameliorer la qualite des prompts
  • Interface CLI - workflow en ligne de commande scriptable pour l'automatisation
  • Interface web - interface navigateur pour des sessions d'evaluation interactives

3 Installation

Necessite Python 3.10 ou superieur.

Option 1 - pip install

  1. Cloner le depot : 
    git clone https://github.com/StealthyLabsHQ/prompting-evaluator.git
cd prompting-evaluator
  2. Installer les dependances : 
    pip install -r requirements.txt

Option 2 - Installation editable

Pour le developpement ou la personnalisation locale :

pip install -e .

4 Configuration

Les cles API peuvent etre definies en variables d'environnement ou stockees dans un fichier .env a la racine du projet. Ollama tourne localement et ne necessite pas de cle API. 

GEMINI_API_KEY=votre_cle_api_google_ai_studio
ANTHROPIC_API_KEY=votre_cle_api_anthropic
Backend Variable Requis
Ollama (local) - Ollama en cours d'execution en local
Google Gemini GEMINI_API_KEY Cle Google AI Studio
Anthropic Claude ANTHROPIC_API_KEY Cle Anthropic Console
Note : Au moins un backend doit etre disponible. Ollama est recommande pour une evaluation hors ligne ou sans frais.

5 Prerequis

  • Python 3.10 ou superieur
  • Au moins un backend :
    • Ollama en local (aucune cle API requise)
    • Cle API Google AI Studio pour Gemini
    • Cle API Anthropic pour Claude
