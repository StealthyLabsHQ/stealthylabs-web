AI Edu Skills
FR

Retour a la documentation

1 Presentation

AI Edu Skills FR est une collection de skills pour ChatGPT et Claude pour accompagner les etudiants francophones dans leur redaction academique et leurs revisions. La couverture va du lycee au master, adaptee aux normes academiques francaises.

Tous les prompts et conseils sont rediges en francais, selon les conventions academiques francaises, incluant les normes AFNOR ainsi que les formats APA et MLA.

2 Fonctionnalites

  • Aide a la redaction academique - structurer dissertations, rapports et memoires selon les normes francaises
  • Preparation aux examens - methodes de revision pour le baccalaureat, BTS, licence et master
  • Recherche documentaire - evaluer les sources, construire des bibliographies, analyser des documents
  • Planification des revisions - calendriers de travail, gestion des priorites, repetition espacee
  • Normes de citation - mise en forme APA, MLA et AFNOR

3 Installation

Skill Claude Code

  1. Ouvrir Claude Code
  2. Executer /skills install ai-edu-skills-FR
  3. Les skills s'activent dans la session en cours

Instructions personnalisees ChatGPT

  1. Ouvrir le depot GitHub
  2. Copier le prompt du skill souhaite
  3. Coller dans les instructions personnalisees ChatGPT ou dans le prompt systeme d'un nouveau GPT

4 Public cible

  • Lyceens - preparation au bac, dissertation, revision des oraux
  • Etudiants BTS / licence - rapports professionnels, redaction academique, gestion des citations
  • Etudiants master - articles de recherche, soutien a la redaction de memoire, gestion de bibliographie
Note : Ce jeu de skills est concu pour les contextes academiques francophones. Pour le soutien en anglais, voir AI Edu Skills.

5 Prerequis

  • Claude - Claude Code avec support des skills, ou Claude.ai
  • ChatGPT - ChatGPT Plus, Team, ou acces API
  • Aucune dependance supplementaire requise
