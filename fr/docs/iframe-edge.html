Vue d'ensemble

iframe-edge est une collection de widgets HTML/CSS/JS tout-en-un, chacun contenu dans un seul fichier. Conçus pour la productivité, les overlays de streaming et plus encore, ils sont spécialement optimisés pour le Corsair Xeneon Edge et le logiciel iCUE.

Aucune dépendance externe — HTML, CSS et JavaScript vanilla. Plug & Play : copiez le code et collez-le.

Fonctionnalités

  • Aucune dépendance externe — HTML, CSS et JavaScript vanilla dans des fichiers uniques
  • Entièrement responsive — s'adapte automatiquement à toute taille de conteneur (M, L, XL)
  • Plug & Play — copiez le code et collez-le
  • Mode sombre OLED / Mode clair avec bascule sur chaque widget
  • Bilingue EN / FR sur tous les widgets
  • Assistant IA pour 3 modèles : Google Gemini, Anthropic Claude, OpenAI ChatGPT

Prérequis

  • Un écran Corsair Xeneon Edge
  • Le logiciel Corsair iCUE installé et à jour

Configuration iFrame dans iCUE

  1. Ouvrez iCUE et allez dans la configuration de votre écran/tableau de bord.
  2. Dans la liste des Widgets (colonne gauche), cliquez sur l'icône </> iFrame.
  3. Choisissez la taille souhaitée : M, L ou XL.
  4. Dans les paramètres iFrame, repérez la zone de texte Code HTML.
  5. Copiez le code <iframe> du tableau ci-dessus et collez-le dans la zone de texte.
Et voilà ! Le widget se chargera directement depuis GitHub Pages.

Assistant IA

L'Assistant IA nécessite une clé API de Google (Gemini), Anthropic (Claude), ou OpenAI (ChatGPT).

Obtenir une clé API

  • Google Gemini (niveau gratuit disponible) — Google AI Studio
  • Anthropic Claude (pay-as-you-go) — Console Anthropic
  • OpenAI ChatGPT (pay-as-you-go) — Plateforme OpenAI

Configuration

  1. Ajoutez le widget dans iCUE avec son code <iframe>.
  2. Cliquez sur l'icône Paramètres en haut à droite du widget.
  3. Sélectionnez votre fournisseur et collez votre clé API.
  4. Cliquez sur Enregistrer. Votre clé est stockée localement dans votre session iCUE.

Spotify Visualizer

Prérequis : Un compte Spotify Premium.

Créer une application Spotify

  1. Allez sur le Spotify Developer Dashboard et connectez-vous.
  2. Cliquez sur Create app.
  3. Remplissez le nom et la description de l'application.
  4. Dans Redirect URL, ajoutez exactement : 
    https://stealthylabshq.github.io/iframe-edge/spotify-visualizer/auth/callback.html
  5. Cochez la case Web API et enregistrez.
  6. Récupérez votre Client ID dans les paramètres de l'application.

Utilisation

  1. Ajoutez le widget dans iCUE avec son URL spécifique.
  2. Au chargement du widget, entrez votre Client ID pour vous connecter.
  3. Vous serez redirigé pour vous connecter et accorder les permissions.
