Arklay
Discord Bot

Retour aux docs

1 Apercu

Arklay est un bot Discord open-source modulaire (v2.0.0) construit avec TypeScript et discord.js v14. Il propose 76 commandes slash reparties en 6 modules independants : lecteur musical, assistant IA, moderation, utilitaires, fun et configuration serveur.

Toutes les donnees sont persistees via SQLite (8 tables) : configuration des serveurs, preferences utilisateur, historique des conversations, avertissements et reprise de la file musicale apres redemarrage. Supporte a la fois les commandes slash et le prefixe texte (.commande ou arklay commande).

Disponible sur github.com/StealthyLabsHQ/arklay-bot.

2 Modules

2.1 Musique (22 commandes)

Lecteur musical multi-source supportant YouTube, Spotify et SoundCloud via yt-dlp. La file reprend automatiquement apres un redemarrage. Embed interactif "En cours de lecture" avec boutons. Deconnexion automatique apres 5 minutes d'inactivite.

2.2 IA (9 commandes)

Systeme double fournisseur : Claude (Anthropic) et Gemini (Google). Selection de modele par utilisateur stockee dans SQLite. L'historique des conversations conserve les 25 derniers messages par utilisateur et par serveur. Limite de debit : cooldown de base 8s, etend a 60s quand l'API est surchargee. Limites journalieres par modele avec multiplicateur proprietaire configurable.

Commande Description
/ask [question] [provider] [lang]Interroger Claude ou Gemini (10 langues supportees)
/summarize [messages] [provider]Resumer les messages recents d'un salon
/tldr [url]Resumer une page web
/nanobanana [prompt] [image]Generer des images avec Gemini (Nano Banana 2)
/setmodel [model]Choisir son modele IA personnel
/translate [language] [text]Traduction assistee par IA
/roast [user]Roast contextuel via IA en analysant les messages recents de la cible
/vision [image] [prompt]Analyser une image avec Claude ou Gemini
/catchupResume IA de l'activite recente du salon

Modeles disponibles :

  • Claude : Sonnet 4.6 (recommande), Opus 4.6 (plus puissant), Haiku 4.5 (plus rapide)
  • Gemini : Flash 3 Preview (recommande), Pro 3.1 Preview (plus puissant), Flash Lite 3.1 (moins cher)

Generation d'images (/nanobanana) :

  • Ratios d'aspect : 1:1, 16:9, 9:16, 4:3, 3:4
  • Resolutions : 512px, 1K, 2K, 4K
  • Styles : photorealistic, anime, cartoon, oil painting, sketch, pixel art, cinematic
  • Cooldown adaptatif qui augmente quand l'API est surchargee

3 Fonctionnalites automatiques

Comportements evenementiels ne necessitant aucune commande :

  • Auto-role - attribue un role a chaque nouveau membre si configure via /config autorole
  • Messages de bienvenue - accueille les nouveaux membres avec un texte personnalisable (variables {user} et {server})
  • Salons vocaux temporaires - rejoindre un salon hub cree un salon personnel, supprime automatiquement quand vide
  • Journalisation des messages - messages supprimes et modifies enregistres dans le salon de logs
  • Auto-moderation IA - analyse de contenu Gemini optionnelle (anti-rebond), enregistre le contenu signale
  • Reprise musicale automatique - la file est automatiquement restauree apres un redemarrage du bot
  • Deconnexion automatique - le bot quitte le vocal apres 5 minutes d'inactivite
  • Suivi des sniped messages - dernier message supprime et modifie suivi par salon

4 Commandes par prefixe texte

Toutes les commandes fonctionnent avec un prefixe texte en plus des commandes slash :

  • Format : .commande [args] ou arklay commande [args]
  • Prefixe par defaut : . (configurable via BOT_PREFIX dans .env)
  • Nom du bot par defaut : arklay (configurable via BOT_NAME dans .env)
  • Exemple : .play never gonna give you up ou arklay ask what is rust
  • Note : les groupes de sous-commandes complexes (/config, /warn, /botrole, /nanobanana) ne sont pas disponibles en prefixe texte

5 Installation

  1. Cloner le depot : 
    git clone https://github.com/StealthyLabsHQ/arklay-bot.git
cd arklay-bot
  2. Installer les dependances : 
    npm install
  3. Copier le fichier d'environnement : 
    cp .env.example .env
  4. Editer .env avec vos tokens et cles API (voir la section Configuration ci-dessous).
  5. (Optionnel) Installer FFmpeg systeme pour une meilleure qualite audio. Sans cela, le ffmpeg-static integre est utilise en secours.
    • Windows : telecharger depuis gyan.dev/ffmpeg/builds
    • Linux : sudo apt install ffmpeg
    • macOS : brew install ffmpeg
  6. Deployer les commandes slash sur Discord : 
    npm run deploy
  7. Demarrer le bot :

    Mode developpement :

    npm run dev

    Ou compiler et lancer en production :

    npm run build
npm run start

Production avec PM2

npm run build
npm run deploy
pm2 start dist/index.js --name arklay-bot
pm2 save

Configuration du portail Discord Developer

  • Aller sur discord.com/developers/applications et creer une application
  • Dans l'onglet Bot, activer : Server Members Intent, Message Content Intent, Presence Intent
  • Dans OAuth2, generer une URL d'invitation avec les scopes : bot, applications.commands
  • Permissions recommandees : Administrator (ou specifiques : Manage Roles, Manage Channels, Kick/Ban Members, Moderate Members, Manage Messages, Connect, Speak)

Optionnel : placer un fichier cookies.txt (format Netscape) a la racine du projet pour contourner la detection de bots YouTube lors de la lecture musicale.

6 Configuration

Toute la configuration se fait via les variables d'environnement dans le fichier .env.

Requises

Variable Description
DISCORD_TOKENToken du bot depuis le portail Discord Developer
CLIENT_IDID de l'application depuis le portail Discord Developer

Optionnelles

Variable Description
GUILD_IDID du serveur de dev pour un deploiement instantane (sans GUILD_ID = deploiement global, jusqu'a 1 heure)
BOT_PREFIXPrefixe des commandes texte (defaut : ".")
BOT_NAMEPrefixe nom du bot pour les commandes texte (defaut : "arklay")
BOT_OWNER_IDID Discord du proprietaire (obtient des limites IA elevees)
BOT_OWNER_MULTIPLIERMultiplicateur de limite de debit proprietaire 0-20 (defaut : 5)
ANTHROPIC_API_KEYAcces direct a l'API Claude (module IA desactive si les deux cles IA sont absentes)
GOOGLE_AI_API_KEYCle API Gemini + generation d'images /nanobanana
GOOGLE_CLOUD_PROJECTProjet GCP pour le routage Claude via Vertex AI
GOOGLE_CLOUD_REGIONRegion GCP pour Vertex AI (defaut : us-east5)
GOOGLE_APPLICATION_CREDENTIALSChemin vers le fichier JSON du compte de service GCP
SPOTIFY_CLIENT_IDResolution des liens Spotify
SPOTIFY_CLIENT_SECRETResolution des liens Spotify
NODE_ENVdevelopment ou production

Si ni ANTHROPIC_API_KEY ni GOOGLE_AI_API_KEY n'est definie, le module IA se desactive automatiquement et toutes les commandes IA seront indisponibles.

7 Prerequis

  • Node.js 20+
  • FFmpeg (installation systeme recommandee ; ffmpeg-static integre utilise en secours)
  • Token de bot Discord (obligatoire)
  • Au moins une cle API IA (ANTHROPIC_API_KEY ou GOOGLE_AI_API_KEY) pour utiliser les commandes IA
