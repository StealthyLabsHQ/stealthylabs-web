Gemini
Discord Bot

Retour aux docs

1 Vue d'ensemble

Gemini Discord Bot est un bot Discord propulse par l'IA Google Gemini. Il permet aux utilisateurs de dialoguer avec Gemini directement depuis n'importe quel serveur Discord via des slash commands, avec support du contexte de conversation entre les messages.

Le bot est leger et concu pour un deploiement simple en auto-hebergement sur tout serveur disposant de Node.js 18+.

2 Fonctionnalites

  • Chat Gemini AI - interagissez avec Google Gemini via des slash commands Discord
  • Contexte de conversation - le bot conserve le contexte au fil d'une conversation
  • Deployable sur serveur - fonctionne sur tout hebergeur ou VPS compatible Node.js
  • Leger - dependances minimales, aucune base de donnees requise

3 Installation

Necessite Node.js 18 ou superieur.

  1. Cloner le depot : 
    git clone https://github.com/StealthyLabsHQ/gemini-discordbot.git
cd gemini-discordbot
  2. Installer les dependances : 
    npm install
  3. Creer un fichier .env et ajouter vos identifiants (voir Configuration).
  4. Demarrer le bot : 
    node index.js

4 Configuration

Toute la configuration se fait via un fichier .env a la racine du projet. Creer le fichier et definir les variables suivantes : 

DISCORD_TOKEN=votre_token_de_bot_discord
GEMINI_API_KEY=votre_cle_api_google_ai_studio
Variable Description
DISCORD_TOKEN Token du bot depuis le Discord Developer Portal
GEMINI_API_KEY Cle API depuis Google AI Studio
Note : Ne jamais commiter le fichier .env dans le controle de version. L'ajouter au .gitignore.

5 Prerequis

  • Node.js 18 ou superieur
  • Application bot Discord - a creer via le Discord Developer Portal
  • Cle API Google AI Studio - disponible sur aistudio.google.com
