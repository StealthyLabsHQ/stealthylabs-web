Retour aux docs

1 Presentation

gemini-mcp-connect est un serveur MCP Python qui connecte Claude Code a Google Gemini, jouant le role d'un agent de deuxieme avis directement dans votre CLI. Il permet de valider des plans, de reviewer du code et de croiser des raisonnements sans quitter le terminal.

Le serveur expose quatre outils accessibles depuis Claude Code : requetes libres, review de code, validation de plan et suivi du quota. Trois niveaux de raisonnement controlent l'equilibre entre rapidite et profondeur d'analyse.

2 Fonctionnalites

  • Agent de deuxieme avis - interrogez Gemini depuis Claude Code
  • Trois niveaux de raisonnement - lite, flash, pro selon les besoins
  • Review de code - critique adaptee au langage via review_code
  • Validation de plan - validez une architecture avant execution via validate_plan
  • Suivi du quota - verifiez les requetes restantes avec gemini_status
  • Installation simple - lancez avec uvx, sans clone necessaire

3 Outils

Le serveur expose quatre outils MCP :

Outil Parametres Description
query_gemini prompt, tier Question libre a Gemini
review_code code, language, tier Review de code adaptee au langage
validate_plan plan, tier Validation d'un plan avant execution
gemini_status - Verification du quota journalier restant

Niveaux de raisonnement

Niveau Vitesse Usage
lite Rapide Recherches rapides, requetes simples
flash Equilibre Review de code et validation generales
pro Raisonnement max Architecture complexe, decisions critiques (100 req/jour)

4 Installation

Option 1 - uvx (recommande)

Aucun clone necessaire. Lancez directement avec uv :

uvx gemini-mcp-connect

Option 2 - Depuis les sources

Necessite Python 3.10+.

git clone https://github.com/StealthyLabsHQ/gemini-mcp-connect.git
cd gemini-mcp-connect
pip install -e .

5 Configuration

Definissez votre cle API Google AI Studio comme variable d'environnement : 

export GEMINI_API_KEY=votre_cle_api

Ajoutez le serveur a votre fichier de configuration MCP de Claude Code (~/.claude/settings.json ou equivalent) : 

{
  "mcpServers": {
    "gemini-mcp-connect": {
      "command": "uvx",
      "args": ["gemini-mcp-connect"],
      "env": {
        "GEMINI_API_KEY": "votre_cle_api"
      }
    }
  }
}
Note : Si vous installez depuis les sources, remplacez "uvx" par "python" et adaptez args pour pointer vers le point d'entree du module.

6 Prerequis

  • Python 3.10+
  • GEMINI_API_KEY - cle a obtenir sur Google AI Studio
  • Claude Code ou tout CLI compatible MCP
  • uv (pour l'installation via uvx)
