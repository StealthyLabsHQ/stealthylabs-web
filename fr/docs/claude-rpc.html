Claude
Rich Presence

Retour aux docs

1 Vue d'ensemble

Claude RPC est un client Discord Rich Presence qui affiche votre activité Claude en temps réel directement sur votre profil Discord. Il détecte automatiquement Claude Code et Claude Desktop, suit le modèle actif, le mode et la durée de session.

Zéro configuration requise – aucun Application ID Discord n'est nécessaire.

2 Fonctionnalités

  • Détection automatique du client – Claude Code ou Claude Desktop
  • Mode Claude Desktop – détecte l'onglet actif (Chat, Cowork, Code)
  • Suivi du modèle en direct – Opus 4.6, Sonnet 4.6, Haiku 4.5, etc.
  • Extended Thinking – affiche quand le mode étendu est actif
  • Détection du contexte 1M – affiche (1M) pour les modèles compatibles
  • Durée de session – liée à votre session réelle
  • Timeout d'inactivité – affiche "Absent" après 15 minutes d'inactivité
  • Barre système – démarrage automatique au boot (Windows)
  • Zéro configuration – aucun Application ID Discord requis
  • Sécurité renforcée – audit de sécurité réussi, toutes les failles corrigées

3 Installation

Option 1 – Exécutable standalone (recommandé)

  1. Téléchargez claude-rpc-windows-x64.zip depuis la dernière release
  2. Extrayez le dossier
  3. Double-cliquez sur claude-rpc.exe
Note : Aucune installation requise – Node.js est inclus dans l'exécutable.

Option 2 – Depuis les sources

Requiert Node.js 18+.

git clone https://github.com/StealthyLabsHQ/claude-rpc.git
cd claude-rpc
npm install

Windows : Double-cliquez sur start-claude.bat

macOS / Linux :

chmod +x start-claude.sh
./start-claude.sh

Ou directement :

node --no-deprecation index.js
