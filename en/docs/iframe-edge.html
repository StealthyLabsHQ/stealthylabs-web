Overview

iframe-edge is a collection of all-in-one HTML/CSS/JS widgets, each contained in a single file. Designed for productivity, streaming overlays, and more, they are specially optimized for the Corsair Xeneon Edge display and iCUE software.

No external dependencies — pure HTML, CSS, and vanilla JavaScript. Plug & Play: copy the code and paste it.

Features

  • No external dependencies — pure HTML, CSS, and vanilla JavaScript in single files
  • Fully Responsive — automatically scales to fit any container size (M, L, XL)
  • Plug & Play — just copy the code and paste it
  • Dark Mode OLED / Light Mode with a toggle on each widget
  • EN / FR bilingual support on all widgets
  • AI Assistant for 3 AI models: Google Gemini, Anthropic Claude, OpenAI ChatGPT

Prerequisites

  • A Corsair Xeneon Edge display
  • Corsair iCUE software installed and up to date

iFrame Setup in iCUE

  1. Open iCUE and go to your screen/dashboard configuration.
  2. In the Widgets list (left column), click on the </> iFrame icon.
  3. Choose your desired size: M, L, or XL.
  4. In the iFrame settings, look for the HTML code text area.
  5. Copy the <iframe> code from the table above and paste it into the text area.
Done! The widget will load live from GitHub Pages.

AI Assistant

The AI Assistant widget requires an API key from either Google (Gemini), Anthropic (Claude), or OpenAI (ChatGPT).

How to get an API key

  • Google Gemini (Free tier available) — Google AI Studio
  • Anthropic Claude (pay-as-you-go) — Anthropic Console
  • OpenAI ChatGPT (pay-as-you-go) — OpenAI Platform

Setup

  1. Add the widget to iCUE using its <iframe> code from the table.
  2. Click the Settings icon in the top right corner of the widget.
  3. Select your preferred provider and paste your API key.
  4. Click Save. Your key is securely stored locally in your iCUE session.

Spotify Visualizer

Prerequisites: A Spotify Premium account.

Create a Spotify app

  1. Go to the Spotify Developer Dashboard and log in.
  2. Click Create app.
  3. Fill in the app name and description.
  4. For the Redirect URL, add exactly: 
    https://stealthylabshq.github.io/iframe-edge/spotify-visualizer/auth/callback.html
  5. Check the Web API box and save.
  6. Find your Client ID in the app settings.

How to use

  1. Add the widget to iCUE with its specific URL.
  2. When the widget loads, enter your Client ID to connect.
  3. You will be redirected to log in and grant permissions.
