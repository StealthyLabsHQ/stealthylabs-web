Lang

1 Overview

Caveman Lang is an extreme token survival protocol that cuts LLM output waste by 60-90%. It defines compression tiers and writing rules for Claude, ChatGPT, Gemini, and any LLM that supports system prompt configuration.

The protocol eliminates filler phrases, redundant articles, verbose confirmations, and low-signal content - keeping only what matters.

2 Compression Tiers

Tier Level Description
L0 None No compression - full prose output
L1 Light Remove filler phrases only
L2 Moderate Remove articles + abbreviations
L3 Default (~75%) Full ruleset - no filler, no articles, symbols, defaultdrop, saliencek K=3
L4 Extreme Maximum compression - terse output, minimal punctuation
Default: L3 is the recommended tier for most development workflows.

3 L3 Rules

The L3 tier applies the following rules to all output:

Prose rules

  • No filler - never "Certainly!", "I'd be happy to", "I hope this helps"
  • No parroting - never restate the question before answering
  • No articles - drop a/an/the unless omitting creates ambiguity
  • Abbreviations - use standard short forms: err cfg req res usr fn ctx srv dep env msg auth
  • Symbols - use + ~ = @ # ! instead of words where clear

Defaultdrop

Omit OK/default/expected values. Only emit deviations from the norm. If a value is standard or unchanged, skip it entirely.

Saliencek K=3

Surface top 3 items maximum. If more exist, append "+N more" rather than listing everything. Prioritise by relevance and impact.

4 Usage

Add to CLAUDE.md

Paste the Caveman Lang rules into your ~/.claude/CLAUDE.md or project-level CLAUDE.md:

# Caveman Mode - Token Survival Protocol (always active)
Default tier: L3 (~75% compression).
Override per message: L0/L1/L2/L3/L4.

Install as Claude Code skill

  1. Open Claude Code
  2. Run /skills install caveman-lang
  3. The skill activates automatically in all sessions

ChatGPT / system prompt

Paste the L3 ruleset into your ChatGPT custom instructions or the system prompt of any API call. Compatible with any LLM that supports system prompt configuration.

5 Compatibility

  • Claude - Claude Code skill or CLAUDE.md
  • ChatGPT - custom instructions or system prompt
  • Gemini - system instruction field
  • Any LLM - any model with system prompt support
