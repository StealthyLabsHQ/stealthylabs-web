1 Overview

Prompting Evaluator is a professional prompt engineering tool designed to evaluate, score, and optimize AI instructions. It supports multiple backends - local models via Ollama and cloud APIs including Google Gemini and Anthropic Claude.

The tool provides both a command-line interface and a web interface, making it suitable for quick terminal-based iteration as well as structured evaluation workflows.

2 Features

  • Multi-model evaluation - run prompts against Ollama, Gemini, and Claude simultaneously
  • Scoring rubrics - structured criteria for consistent prompt assessment
  • Prompt comparison - side-by-side analysis of prompt variants
  • Optimization suggestions - actionable recommendations to improve prompt quality
  • CLI interface - scriptable command-line workflow for automation
  • Web interface - browser-based UI for interactive evaluation sessions

3 Installation

Requires Python 3.10 or higher.

Option 1 - pip install

  1. Clone the repository: 
    git clone https://github.com/StealthyLabsHQ/prompting-evaluator.git
cd prompting-evaluator
  2. Install dependencies: 
    pip install -r requirements.txt

Option 2 - Editable install

For development or local customization:

pip install -e .

4 Configuration

API keys can be set as environment variables or stored in a .env file at the project root. Ollama runs locally and requires no API key. 

GEMINI_API_KEY=your_google_ai_studio_api_key
ANTHROPIC_API_KEY=your_anthropic_api_key
Backend Variable Required
Ollama (local) - Ollama running locally
Google Gemini GEMINI_API_KEY Google AI Studio key
Anthropic Claude ANTHROPIC_API_KEY Anthropic Console key
Note: At least one backend must be available. Ollama is recommended for offline or cost-free evaluation.

5 Requirements

  • Python 3.10 or higher
  • At least one backend:
    • Ollama running locally (no API key needed)
    • Google AI Studio API key for Gemini
    • Anthropic API key for Claude
