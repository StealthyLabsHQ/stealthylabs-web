Arklay
Discord Bot

Back to docs

1 Overview

Arklay is an open-source modular Discord bot (v2.0.0) built with TypeScript and discord.js v14. It provides 76 slash commands across 6 independent modules: music player, AI assistant, moderation, utility, fun, and server configuration.

All state persists via SQLite (8 tables): guild configuration, user preferences, conversation history, warning records, and music queue resume across bot restarts. Supports both slash commands and text prefix (.command or arklay command).

Available at github.com/StealthyLabsHQ/arklay-bot.

2 Modules

2.1 Music (22 commands)

Multi-source music player supporting YouTube, Spotify, and SoundCloud via yt-dlp. Queue auto-resumes across bot restarts. Interactive Now Playing embed with buttons. Auto-disconnects after 5 minutes of inactivity.

2.2 AI (9 commands)

Dual provider system: Claude (Anthropic) and Gemini (Google). Per-user model selection stored in SQLite. Conversation history retains the last 25 messages per user per guild. Rate limiting applies an 8s base cooldown that extends to 60s when the API is busy. Daily usage limits apply per model, with a configurable owner multiplier.

Command Description
/ask [question] [provider] [lang]Ask Claude or Gemini (10 languages supported)
/summarize [messages] [provider]Summarize recent channel messages
/tldr [url]Summarize a webpage
/nanobanana [prompt] [image]Generate images with Gemini (Nano Banana 2)
/setmodel [model]Choose personal AI model
/translate [language] [text]AI-powered translation
/roast [user]Contextual AI roast using target's recent messages
/vision [image] [prompt]Analyze an image with Claude or Gemini
/catchupAI-powered summary of recent channel activity

Available models:

  • Claude: Sonnet 4.6 (recommended), Opus 4.6 (most powerful), Haiku 4.5 (fastest)
  • Gemini: Flash 3 Preview (recommended), Pro 3.1 Preview (most powerful), Flash Lite 3.1 (cheapest)

Image generation (/nanobanana):

  • Aspect ratios: 1:1, 16:9, 9:16, 4:3, 3:4
  • Resolutions: 512px, 1K, 2K, 4K
  • Styles: photorealistic, anime, cartoon, oil painting, sketch, pixel art, cinematic
  • Adaptive cooldown increases when API is busy

3 Automatic Features

Event-driven behaviors that require no slash command:

  • Auto-role - assigns a role to every new member when configured via /config autorole
  • Welcome messages - greets new members with customizable text ({user} and {server} placeholders)
  • Temporary voice channels - joining a hub channel creates a personal VC, auto-deleted when empty
  • Message logging - deleted and edited messages logged to mod channel
  • AI auto-moderation - optional Gemini content scanning (debounced), logs flagged content
  • Music auto-resume - queue automatically restores after bot restart
  • Music auto-disconnect - bot leaves voice after 5 minutes of inactivity
  • Snipe tracking - last deleted and edited messages tracked per channel

4 Text Prefix Commands

All commands work with text prefix in addition to slash commands:

  • Format: .command [args] or arklay command [args]
  • Default prefix: . (configurable via BOT_PREFIX in .env)
  • Default bot name: arklay (configurable via BOT_NAME in .env)
  • Example: .play never gonna give you up or arklay ask what is rust
  • Note: complex subcommand groups (/config, /warn, /botrole, /nanobanana) are not available as text prefix

5 Installation

  1. Clone the repository: 
    git clone https://github.com/StealthyLabsHQ/arklay-bot.git
cd arklay-bot
  2. Install dependencies: 
    npm install
  3. Copy the environment template: 
    cp .env.example .env
  4. Edit .env with your tokens and API keys (see the Configuration section below).
  5. (Optional) Install system FFmpeg for best audio quality. Without it, the bundled ffmpeg-static is used as fallback.
    • Windows: download from gyan.dev/ffmpeg/builds
    • Linux: sudo apt install ffmpeg
    • macOS: brew install ffmpeg
  6. Deploy slash commands to Discord: 
    npm run deploy
  7. Start the bot:

    Development mode:

    npm run dev

    Or build and start for production:

    npm run build
npm run start

Production with PM2

npm run build
npm run deploy
pm2 start dist/index.js --name arklay-bot
pm2 save

Discord Developer Portal setup

  • Go to discord.com/developers/applications and create an application
  • Under Bot, enable: Server Members Intent, Message Content Intent, Presence Intent
  • Under OAuth2, generate an invite URL with scopes: bot, applications.commands
  • Recommended permissions: Administrator (or specific: Manage Roles, Manage Channels, Kick/Ban Members, Moderate Members, Manage Messages, Connect, Speak)

Optional: place a cookies.txt (Netscape format) in the project root to bypass YouTube bot detection for music playback.

6 Configuration

All configuration is done via environment variables in the .env file.

Required

Variable Description
DISCORD_TOKENBot token from Discord Developer Portal
CLIENT_IDApplication ID from Discord Developer Portal

Optional

Variable Description
GUILD_IDDev server ID for instant command deployment (no GUILD_ID = global deployment, up to 1 hour)
BOT_PREFIXText command prefix (default: ".")
BOT_NAMEBot name prefix for text commands (default: "arklay")
BOT_OWNER_IDOwner's Discord user ID (gets boosted AI rate limits)
BOT_OWNER_MULTIPLIEROwner rate limit multiplier 0-20 (default: 5)
ANTHROPIC_API_KEYDirect Claude API access (AI module disabled if both AI keys absent)
GOOGLE_AI_API_KEYGemini API key + /nanobanana image generation
GOOGLE_CLOUD_PROJECTGCP project for Vertex AI Claude routing
GOOGLE_CLOUD_REGIONGCP region for Vertex AI (default: us-east5)
GOOGLE_APPLICATION_CREDENTIALSPath to GCP service account JSON
SPOTIFY_CLIENT_IDSpotify link resolution
SPOTIFY_CLIENT_SECRETSpotify link resolution
NODE_ENVdevelopment or production

If neither ANTHROPIC_API_KEY nor GOOGLE_AI_API_KEY is set, the AI module auto-disables gracefully and all AI commands will be unavailable.

7 Requirements

  • Node.js 20+
  • FFmpeg (system install recommended; bundled ffmpeg-static used as fallback)
  • Discord bot token (required)
  • At least one AI API key (ANTHROPIC_API_KEY or GOOGLE_AI_API_KEY) to use AI commands
