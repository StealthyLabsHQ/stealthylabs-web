Specter
Discord Bot

Back to docs

1 Overview

Specter is a modular Discord bot (v2.0.0) built with TypeScript and discord.js v14. It provides 74 slash commands across 6 independent modules: music player, AI assistant, moderation, utility, fun, and server configuration.

All state persists via SQLite (8 tables): guild configuration, user preferences, conversation history, warning records, and music queue resume across bot restarts. Supports both slash commands and text prefix (.command or specter command).

2 Modules

2.2 AI (9 commands)

Dual provider system: Claude (Anthropic) and Gemini (Google). Per-user model selection stored in SQLite. Conversation history retains the last 25 messages per user per guild. Rate limiting applies an 8s base cooldown that extends to 60s when the API is busy. Daily usage limits apply per model.

Command Description
/ask [question] [provider] [lang]Ask Claude or Gemini
/summarize [messages] [provider]Summarize recent channel messages
/tldr [url]Summarize a webpage
/nanobanana [prompt] [style] [ratio] [resolution]Generate images with Gemini
/setmodel [model]Choose personal AI model
/translate [language] [text]AI-powered translation
/roast [user]Fun AI roast using user's recent messages
/vision [image] [prompt]Analyze an image with AI
/catchupSummarize recent conversation context

Available models:

  • Claude: Sonnet 4.6 (recommended), Opus 4.6 (most powerful), Haiku 4.5 (fastest)
  • Gemini: Flash 3 Preview (recommended), Pro 3.1 Preview (most powerful), Flash Lite 3.1 (cheapest)

Image generation (/nanobanana):

  • Styles: photorealistic, anime, cartoon, oil painting, sketch, pixel art, cinematic
  • Aspect ratios: 1:1, 16:9, 9:16, 4:3, 3:4
  • Resolutions: 512px, 1K, 2K, 4K (per-tier daily limits)
  • Adaptive cooldown increases when API is busy

3 Automatic Features

Event-driven behaviors that require no slash command:

  • Auto-role - assigns a role to every new member when configured via /config autorole
  • Welcome messages - greets new members in a designated channel
  • Temporary voice channels - joining a hub channel creates a personal VC, auto-deleted when empty
  • Message logging - deleted and edited messages logged to mod channel
  • AI auto-moderation - optional Gemini content scanning (5s debounce per user), flags and logs suspicious content
  • Message snipe - tracks last deleted/edited message per channel

4 Text Prefix Commands

All commands work with text prefix in addition to slash commands:

  • Format: .command [args] or specter command [args]
  • Example: .play never gonna give you up or specter ask what is rust
  • Prefix and bot name are configurable via .env
