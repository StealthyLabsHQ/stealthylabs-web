Specter
Discord Bot

Back to docs

1 Overview

Specter is a modular Discord bot built with TypeScript and discord.js v14, combining music playback, AI-powered commands (Claude + Gemini), image generation, server moderation, and utility tools. It provides 61 slash commands across 6 independent modules.

SQLite handles all persistence: guild configuration, user preferences, conversation history, and music queue resume across bot restarts. Version 2.0.0.

2 Modules

2.2 AI (8 commands)

Dual provider system: Claude (Anthropic) and Gemini (Google). Per-user model selection stored in SQLite. Conversation history retains the last 25 messages per user per guild. Rate limiting applies an 8s base cooldown that extends to 60s when the API is busy. Daily usage limits apply per model.

Command Description
/ask [question] [provider] [lang]Ask Claude or Gemini with language selection
/summarize [messages] [provider]Summarize recent channel messages
/nanobanana [prompt] [style] [ratio] [resolution]Generate images with Gemini
/setmodel [model]Choose personal AI model (Claude or Gemini variants)
/translate [language] [text]AI-powered translation
/roast [user]Fun AI roast of a user
/vision [image] [prompt]Analyze an image with AI
/catchupSummarize recent conversation context

Available models:

  • Claude: Sonnet 4.6 (recommended), Opus 4.6 (most powerful), Haiku 4.5 (fastest)
  • Gemini: Flash 3 Preview (recommended), Pro 3.1 Preview (most powerful), Flash Lite 3.1 (cheapest)

Image generation (/nanobanana):

  • Styles: photorealistic, anime, cartoon, oil painting, sketch, pixel art, cinematic
  • Aspect ratios: 1:1, 16:9, 9:16, 4:3, 3:4
  • Resolutions: 512px, 1K, 2K, 4K (per-tier daily limits)
  • Adaptive cooldown increases when API is busy

3 Automatic Features

Event-driven behaviors that require no slash command:

  • Auto-role - assign a role to every new member (requires /config autorole)
  • Welcome messages - greet new members in a designated channel
  • Temporary voice channels - joining a hub channel creates a personal voice channel, deleted when empty
  • Message logging - deleted and edited messages logged to mod channel
  • AI auto-moderation - optional Gemini content scanning with 5s debounce per user, logs flagged content

4 Installation

  1. Clone the repository and enter the directory: 
    git clone <repository-url>
cd discord-bot
  2. Install dependencies: 
    npm install
  3. Copy the environment template: 
    cp .env.example .env
  4. Edit .env with your tokens and API keys.
  5. Deploy slash commands to Discord: 
    npm run deploy
  6. Start in development mode: 
    npm run dev
    Or build and run for production: 
    npm run build
npm run start

Production with PM2

npm run build
npm run deploy
pm2 start dist/index.js --name specter
pm2 save

GOOGLE_APPLICATION_CREDENTIALSPath to GCP service account JSON
SPOTIFY_CLIENT_IDSpotify link resolution
SPOTIFY_CLIENT_SECRETSpotify link resolution
NODE_ENVdevelopment or production

Discord Developer Portal

Required settings in the Discord Developer Portal:

  • Privileged intents: Server Members, Message Content, Guild Members, Guild Presences
  • Bot scopes: bot, applications.commands
  • Recommended permissions: Administrator, or granular: Manage Roles, Manage Channels, Kick/Ban Members, Moderate Members, Manage Messages
Note: Place a cookies.txt (Netscape format) in the project root to bypass YouTube bot detection for music.

6 Requirements

  • Node.js 20+
  • FFmpeg - system installation recommended; ffmpeg-static used as fallback
  • At minimum DISCORD_TOKEN + CLIENT_ID to run - AI and music modules gracefully degrade without optional API keys
