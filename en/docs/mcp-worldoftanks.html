1 Overview

mcp-worldoftanks is a TypeScript MCP server that connects AI CLIs to the Wargaming.net API, enabling natural language queries over World of Tanks account statistics. Ask your AI assistant to pull player profiles, tank performance data, or battle history directly from your terminal.

Compatible with Claude Code, Claude Desktop, Gemini CLI, and any other MCP-compatible AI client.

2 Features

  • Account statistics - overall win rate, damage, WN8, and more
  • Tank statistics - per-vehicle performance breakdown
  • Player profiles - account details and clan info
  • Battle history - recent session data
  • Multi-client support - works with any MCP-compatible AI CLI
  • Wargaming.net API - official data source, all regions supported

3 Installation

Option 1 - npx (no build required)

Add directly to your MCP config using npx. See the Configuration section below.

Option 2 - From source

Requires Node.js 18+.

git clone https://github.com/StealthyLabsHQ/mcp-worldoftanks.git
cd mcp-worldoftanks
npm install
npm run build

4 Configuration

Set your Wargaming.net API key as an environment variable: 

export WG_API_KEY=your_api_key_here

Add the server to your MCP configuration file. Using npx: 

{
  "mcpServers": {
    "mcp-worldoftanks": {
      "command": "npx",
      "args": ["-y", "@stealthylabs/mcp-worldoftanks"],
      "env": {
        "WG_API_KEY": "your_api_key_here"
      }
    }
  }
}

Or if installed from source, point to the built output:

{
  "mcpServers": {
    "mcp-worldoftanks": {
      "command": "node",
      "args": ["/path/to/mcp-worldoftanks/dist/index.js"],
      "env": {
        "WG_API_KEY": "your_api_key_here"
      }
    }
  }
}
API key: Obtain a free Wargaming.net API key at developers.wargaming.net.

5 Requirements
