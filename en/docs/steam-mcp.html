1 Overview

steam-mcp is a TypeScript MCP server that exposes the Steam Web API to any MCP-compatible AI client. Query player profiles, browse game libraries, check achievements, look up VAC ban status, search the store, and more - all through natural language from your terminal.

Eight slash commands cover the most common Steam data operations, from profile lookups to co-op game discovery and playtime trend analysis.

2 Features

  • Player profiles - public Steam profile data
  • Game library - owned games and playtime
  • Achievements - per-game achievement progress
  • Game details - store information for any app
  • VAC ban status - check ban records for any Steam ID
  • Co-op discovery - find games shared between players
  • Playtime trends - analyze recent activity patterns
  • Raw API access - direct Steam Web API endpoint queries

3 Commands

The server exposes 8 slash commands:

Command Description
/steam:profile Fetch a Steam player profile
/steam:library List owned games and playtime
/steam:achievements Get achievement progress for a game
/steam:game Retrieve store details for an app
/steam:vac Check VAC and game ban status
/steam:coop Find games shared between multiple players
/steam:trend Analyze recent playtime trends
/steam:api Direct access to any Steam Web API endpoint

4 Installation

Option 1 - npx (no build required)

Add directly to your MCP config using npx. See the Configuration section below.

Option 2 - From source

Requires Node.js 18+.

git clone https://github.com/StealthyLabsHQ/steam-mcp.git
cd steam-mcp
npm install
npm run build

5 Configuration

Set your Steam API key as an environment variable: 

export STEAM_API_KEY=your_api_key_here

Add the server to your MCP configuration file. Using npx: 

{
  "mcpServers": {
    "steam-mcp": {
      "command": "npx",
      "args": ["-y", "@stealthylabs/steam-mcp"],
      "env": {
        "STEAM_API_KEY": "your_api_key_here"
      }
    }
  }
}

Or if installed from source, point to the built output:

{
  "mcpServers": {
    "steam-mcp": {
      "command": "node",
      "args": ["/path/to/steam-mcp/dist/index.js"],
      "env": {
        "STEAM_API_KEY": "your_api_key_here"
      }
    }
  }
}
API key: Obtain a free Steam Web API key at steamcommunity.com/dev/apikey. A Steam account is required.

6 Requirements

  • Node.js 18+
  • STEAM_API_KEY environment variable - obtain at steamcommunity.com/dev/apikey
  • Claude Code, Claude Desktop, or any MCP-compatible AI client
