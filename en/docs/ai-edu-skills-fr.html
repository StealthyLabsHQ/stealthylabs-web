AI Edu Skills
FR

Back to docs

1 Overview

AI Edu Skills FR is a collection of skills for ChatGPT and Claude designed to support French-speaking students in academic writing and revision. Coverage spans from high school to master's level, adapted to French academic norms.

All prompts and guidance are written in French, following French academic conventions including AFNOR citation standards alongside APA and MLA formats.

2 Features

  • Academic writing support - structuring essays, dissertations, and reports following French norms
  • Exam preparation - baccalaureat, BTS, licence, and master-level revision methods
  • Document research - evaluating sources, building bibliographies, documentary analysis
  • Revision planning - study schedules, priority management, spaced repetition
  • Citation standards - APA, MLA, and AFNOR formatting guidance

3 Installation

Claude Code skill

  1. Open Claude Code
  2. Run /skills install ai-edu-skills-FR
  3. The skills activate in the current session

ChatGPT custom instructions

  1. Open the GitHub repository
  2. Copy the desired skill prompt
  3. Paste into ChatGPT custom instructions or a new GPT system prompt

4 Target Audience

  • Lycee students - bac preparation, essay writing, oral revision
  • BTS / licence students - professional reports, academic writing, citation management
  • Master students - research papers, thesis support, bibliography management
Note: This skill set is designed for French-language academic contexts. For English-language academic support, see AI Edu Skills.

5 Requirements

  • Claude - Claude Code with skill support, or Claude.ai
  • ChatGPT - ChatGPT Plus, Team, or API access
  • No additional dependencies required
